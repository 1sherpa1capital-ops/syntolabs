# Synto Labs

Personal workspace and agency landing pages.

## Projects

- **Landing Page**: [syntolabs.xyz](https://syntolabs.xyz) - Core agency site.
- **My Page**: [rhigden.vercel.app](https://rhigden.vercel.app) - Personal portfolio and technical showcase.

## Tech Stack

- **React 19**
- **Vite 6**
- **Tailwind CSS v4**
- **Framer Motion**
- **Three.js** (Landing Page)
- **GSAP** (Landing Page)

## Commands

```bash
# Install dependencies (from subdirectories)
bun install

# Development
bun run dev

# Build
bun run build

# Deploy (from subdirectories via Vercel CLI)
vercel --prod
```

## Deployment

Managed via Vercel. Each project is a separate Vercel deployment linked to its respective subdirectory.
